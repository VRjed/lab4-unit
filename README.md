# Лабораторная работа №4
## Написание Unit-тестов

### Описание
Проект содержит простой модуль на языке Python для вычисления площади и периметра прямоугольника,
а также набор unit-тестов для проверки корректности работы функций.

### Состав проекта
- rectangle.py — модуль с реализуемыми функциями
- test_rectangle.py — unit-тесты на основе unittest

### Запуск тестов
Для запуска unit-тестов необходимо выполнить команду:

python -m unittest discover

### Используемые технологии
- Python
- unittest
